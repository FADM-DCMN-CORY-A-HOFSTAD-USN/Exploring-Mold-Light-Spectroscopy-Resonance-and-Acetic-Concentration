This repository serves as a personal reference library and documentation archive exploring theoretical analytical workflows. It focuses on the non-destructive identification of organic structures using light spectroscopy, vibrational resonance, and variable chemical solvents.

* **Resonance Tuning:** Tracking the vibrational and rotational kinetic energy forms within chemical bonds when exposed to fixed target frequencies.
* **Acetic Solvent Profiling:** Modeling how changes in acetic acid concentration modify surface structures and interfere with cellular identification metrics.
